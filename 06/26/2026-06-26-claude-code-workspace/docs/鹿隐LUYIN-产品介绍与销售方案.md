## 品牌故事

大兴安岭北麓，根河市敖鲁古雅乡。

这里是中国的冷极，也是鄂温克人最后的驯鹿部落。三百年前，他们从贝加尔湖一路向东，带着驯鹿走进这片白桦林。到今天，真正的使鹿鄂温克只剩不到 30 人。

他们不建房子，住在撮罗子里——一个锥形的桦树皮帐篷。他们搬家时不带走任何多余的东西，一个桦树皮箱子就是全部家当。他们在白桦树上刻标记，不是占山为王，只是告诉后面的人：往这边走，有水。他们相信每一棵树、每一头鹿、每一阵风里都有灵。

**鹿隐**做的不是文创产品。我们是把鄂温克人留在森林里的那些痕迹，做成你可以带走的东西。

每件产品都来自一个真实的鄂温克生活动作——不是贴上去的文案，是猎人刻在树上的标记、是驯鹿卧下时的安心、是桦树皮箱子里的全部世界。

### 产品一：「白桦林日记」A6 手绘线圈本

**画面**：封面是一幅手绘——驯鹿穿过晨雾中的白桦林。

鄂温克人的一天从驯鹿开始。清晨，猎人推开撮罗子的门，鹿群已经在白桦林里啃苔藓。他数了数——二十三头，都在。然后他回屋里削一块桦树皮，开始新的一天。

这个封面就是那个画面。四种可选：**晨牧**（迷雾中的鹿影）、**迁徙**（鹿群过河）、**撮罗子炊烟**（家的信号）、**星野下的鹿铃**（夜晚的森林）。

**卖点**：A6 巴掌大小，线圈装订可以完全平摊。内页空白，不替你做任何决定——写什么、画什么、贴什么，是你的事。

**消费者**：想买点什么但不想花太多钱的人。景区游客、学生、手账入门者。

### 产品二：「猎人的标记」和纸胶带 ×2

**不是装饰品，是导航工具。**

鄂温克猎人走进大兴安岭之前，会先在白桦树上刻一道标记。指向猎物出没的方向，标注水源的位置，提醒后面的同伴归途。这是森林里的 Google Maps。

两卷胶带：一卷叫**「方向」**——鹿角纹样循环，撕下来贴在纸上，鹿角指向哪里，就是你今天要去的地方。一卷叫**「记号」**——白桦眼纹循环，每一棵白桦树都有自己的眼睛，你贴下的每一段都是你今天留下的一个锚点。

**卖点**：日本和纸材质，手撕不断，贴错可揭。但真正卖的是使用方式——它不是让你"装饰"手账，是让你像猎人一样"标记"你的一天。方向卷贴在你出发的地方，记号卷贴在你到达的地方。

**消费者**：手账重度用户、喜欢"功能叙事"的人——他们不爱纯装饰，他们要每一段胶带都有贴它的理由。

### 产品四：「大兴安岭的呼吸」桦木香薰蜡烛

**气味，是一个地方的终极记忆。**

调香师去了大兴安岭的秋天，然后把那个气味带回来：白桦树汁的清甜（前调，像刚剥开的树皮），松针的凛冽（中调，像穿过一片松林时风打在脸上），苔藓和湿泥土的潮湿感（后调，像踩在森林地表，脚陷下去一点点）。

木质烛芯。燃烧时会发出轻微的噼啪声——不是那种吵闹的爆裂，是篝火的节奏，每秒两三下。你把它放在书桌上，不用盯着看，耳朵会告诉你它在哪里。

**容器**是一只哑光陶瓷杯。桦皮白 / 苔原绿两色可选。杯身刻了一道极简的鹿角线稿——不是抢眼的装饰，是你不经意转动手腕时才注意到的细节。

**卖点**：它不是"闻起来像森林"——那太廉价了。它是"大兴安岭深秋的上午，你在白桦林里站了一会儿"。气味有层次，燃烧有声音，杯子烧完可以继续用。

**消费者**：香薰爱好者、在家工作的人、送朋友暖居礼物的人。尤其适合在小红书做开箱——木质烛芯的噼啪声是天然的 ASMR 素材。

### 产品五：「撮罗子的早晨」几何鹿角杯

**一只杯子。但你握着它的时候，故事就开始了。**

杯身印了一头卧鹿的极简线条。位置有意放在中下部——你握住杯子的时候，虎口刚好盖住鹿身，只露出鹿角。

这个设计来自一句鄂温克谚语：「驯鹿卧下，就是家。」

鄂温克人跟着驯鹿迁徙。驯鹿在哪里卧下，他们就在哪里搭撮罗子。所以当你的手握紧这只杯子——你手心的温度覆盖在卧鹿身上——它的身体是暖的，和你一样的温度。而它的角指向外面，指向今天。

**卖点**：350ml 日常容量，哑光釉面不滑手。桦皮白 / 苔原绿两色。最核心的销售内容不是杯子本身，是"握"这个动作——需要拍一条手持杯子的视频：手指张开→握住→鹿身消失→鹿角露出。

**消费者**：杯控、日常喝咖啡/茶的人、情侣礼物（一人一只，一白一绿）。

**这不是一个图案，是一张地图。**

鄂温克人一年搬家四次。春天去河边，夏天上高地，秋天进密林，冬天回山脚。每次搬家，整个部落一起走——驯鹿驮着桦树皮箱子，猎人走在最前面，女人和孩子在中间，老人骑着鹿在最后。

帆布袋上印的，就是这条迁徙路线——用抽象几何线条画的：河流是苔原绿的曲线，山脉是深木棕的三角，营地是暖驼色的圆点，虚线串联起春夏秋冬四个落脚处。

袋子内侧有个小口袋，印了一句话：「家是移动的，森林是永恒的。」

**卖点**：16 安厚帆布，耐操。但卖的不是材质——是这句话。背这个袋子出门，你也是在迁徙。从这个咖啡馆到那个书店，从家到公司，从这座城市到那座城市。鄂温克人搬了一辈子家，他们知道——家不是墙和屋顶，是跟你一起移动的那些东西。

**消费者**：帆布袋收集者、日常通勤、旅行随身。适合搭配「迁徙」的故事做内容——"今天背着我的迁徙地图去了三个地方"。

### 产品九：「森林的容器」桦树皮手工盒

**真正的手工。真正的桦树皮。**

由还在敖鲁古雅做手艺的鄂温克匠人制作。工序：春天剥树皮（不能伤到树干）→ 蒸煮软化 → 压平晾干 → 裁切 → 皮绳手工缝合。没有两个盒子是一样的——每一块桦树皮的纹理、颜色、疤痕分布都不同。

鄂温克人用桦树皮做一切容器：船（可以坐三个人过河）、箱子（搬家时装所有家当）、摇篮（新生儿的第一张床）。这是一个有 300 年历史的传统。这个盒子是微缩版——放笔墨、放首饰、放你最重要的那几样东西。

盒内印了一句话：「鄂温克人用桦树皮做船、做容器、做摇篮。这个盒子是森林的容器，装的什么由你决定。」

**卖点**：真手工艺，不可复制。每个盒子的纹理是不同的——这不是瑕疵，是这件产品最值钱的地方。限量生产，不追求标准化。

**消费者**：手工艺收藏者、喜欢独一无二之物的人、送重要礼物的人。

### 产品十：「皮绳上的四季」植鞣皮手账

**会和你一起老去。**

意大利植鞣皮，手工染成暖驼色——和驯鹿皮毛一样的颜色。封面压印了鹿角轮廓。

植鞣皮的特点：它会变色。阳光照过的地方会变深，手摸过的地方会变亮，放在桌上不动的地方会保持原色。用一个月，它就是你的了——上面的每一道痕迹都是你的使用印记。

可替换内芯。这本手账不是用一年就扔的——它跟一辈子。

扉页印了一句话：「鄂温克人相信万物有灵。一本跟你一起老去的手账，是你的灵。」

**卖点**：时间叙事。不卖"今天它长这样"，卖"一年后它长什么样"。可以做一条对比视频：崭新的皮手账 vs 用了一年的皮手账——变色、包浆、痕迹。那条视频本身就是最好的广告。

**消费者**：植鞣皮/手工皮具爱好者、终身手账使用者、给自己买一件能陪伴很久的东西的人。

### 产品十二：「让森林进来」白桦扩香木

**不用电，不加热。森林自己走进来。**

一整块桦木实木，切成撮罗子的锥形——就是鄂温克人的帐篷。顶部有个小凹槽。滴 3-5 滴精油（白桦+雪松+杜松子），精油沿着木纹慢慢渗透，然后挥发。整个过程是无声、无火、无电的。

15 分钟后，你房间的空气变了。不那么像房间了，更像大兴安岭的秋天。

配一瓶 5ml 复方精油，大概能用两三个月。

**卖点**：极简的仪式感。不是香薰机那种"我在用香薰"的刻意感，是一块木头安静地放在那里，你都快忘了，然后某天进门的时候突然闻到——就是那个瞬间。

**消费者**：香薰进阶用户（已经过了蜡烛阶段）、极简主义者、办公室桌面用。

### 产品十三：「最后的森林」限量艺术微喷

**邀请一位插画师，去敖鲁古雅住一周。画她看到的。**

限量 100 版，手写编号，卖完永不加印。

画面内容由插画师决定。可能是驯鹿群穿越冻土，可能是老人在撮罗子里削木头的背影，可能是秋天白桦林里的一束光。这些都不预设——插画师的真实所见比我们的想象更有价值。

博物馆级无酸纸，艺术微喷（Giclée），100 年不褪色。

附收藏证书——设计成鄂温克"森林通行证"风格：画作缩略图、插画师签名、限量编号、鹿隐钢印、日期。

**卖点**：稀缺性和真实性。不是"我们设计了一张好看的画"，而是"一个创作者真的去了那里，这是她看到的"。100 版的限量意味着它卖完就没有了——这是最早卖完的产品。

**消费者**：艺术爱好者、限量版收藏者、想要一件"不只是消费品"的东西的人。

- **线上限量发售**（占比 80%）：淘宝/小红书/自有小程序。每件产品标注限量编号。预售制——先卖再产，控制库存风险。
- **敖鲁古雅本地**（占比 20%）：少量铺在根河/敖鲁古雅的高端民宿和在地文化空间。来都来了，买一件真正的桦树皮手工制品回家。

**限量叙事**：「根河」整个系列都是限量逻辑——桦树皮盒（手艺人产能有限）、皮手账（皮料批次不同，每批颜色微差）、艺术微喷（100 版）。不追求规模化。卖完一批等下一批。这种"缺货感"本身就是高端线的品牌资产。

## 包装与开箱

不展开逐条讲了，核心是：**盒子不是容器，是故事的第一页。**

三层包装对应三种"打开"：

- **遇见** —「猎人的行囊」：牛皮纸盒 + 桦树皮腰封，撕开腰封 = 猎人在树上刻下第一道标记。盒底展开是敖鲁古雅手绘地图。
- **日常** —「鄂温克人的桦皮箱」：纹理硬盒 + 皮绳捆扎，解开皮绳 = 在营地安顿下来。盒子可以在用完后继续当收纳。
- **根河** —「萨满的宝藏匣」：真桦树皮贴面 + 火漆封印 + 四层递进。每层之间夹一张纸条：「你听见鹿铃了」「脚下是苔藓，很软」「抬起头，你已经在森林深处」「这个箱子本该装驯鹿铃铛和茶叶。现在你装什么，它就是什么。」

所有包装**全程无塑料**。皮绳、麻绳、木丝、牛皮纸、亚麻布。每盒附带一小束大兴安岭干苔藓：「这是驯鹿的食物，也是你脚下森林的碎片。」

## 销售渠道策略

### 渠道一：小红书（核心战场）

文创产品在小红书的购买链路是：**看见一个故事 → 产生情感连接 → 点进商品页 → 下单**。

内容策略不是"种草"，是"讲故事"：
- 「猎人的标记」胶带 → 标题：「鄂温克猎人的森林导航系统，被我贴在日记本上了」
- 「撮罗子的早晨」杯子 → 标题：「握住杯子时鹿身会消失，这是有原因的设计」
- 「你是谁？」胸针 → 标题：「测测你的森林身份——领路的、筑巢的、还是生长的？」
- 「皮绳上的四季」皮手账 → 标题：「一本会跟你一起老去的手账（对比：用了 3 个月 vs 全新的）」

每条内容的核心是**展示产品带来的那个瞬间**——握杯子的手势、撕胶带的动作、摸手账封面的停顿、蜡烛燃烧时噼啪声的白噪音。

**账号矩阵**：品牌号（鹿隐 LUYIN）+ 合作 KOC（手账博主、香薰博主、生活方式博主、读书博主）——每个品找对应垂类的博主。

### 渠道二：淘宝/有赞

- 遇见系列 → 引流款，低客单价拉新客，关联推荐日常系列
- 日常系列 → 主力利润款，商品详情页用故事而非参数
- 根河系列 → 限量预售，详情页放手工制作过程的视频

**商品详情页结构**（以「迁徙」帆布袋为例）：
1. 首图：帆布袋上身图（森林/城市街景背景）
2. 一句话：「鄂温克人一年搬家四次。这条路线，被印在了袋子上。」
3. 迁徙故事（100 字）
4. 图案细节特写（地图符号解读）
5. 材质/尺寸/洗涤说明（放在最后，只有真正想买的人才滑到这里）

### 渠道三：线下

- **敖鲁古雅/根河/呼伦贝尔**：景区店、游客中心、机场。主推遇见系列（冲动消费）+ 高端系列（来都来了）。日常系列铺少量样品，引导线上下单。
- **独立书店/买手店**：选日常系列。陈列方式不是货架——是一个"鄂温克人的桦皮箱"放在桌上，顾客自己解开皮绳打开看。店员不用推销，开箱体验自己说话。
- **市集/快闪**：周末市集、文创展。全系列展示 + 现场体验（摸桦树皮、闻蜡烛、握杯子、试胸针）。在现场下单送限定赠品（干苔藓小袋）。

## 目标消费者画像

### 画像一：好奇心驱动的旅游者
- 到了敖鲁古雅/呼伦贝尔，想带点什么回去
- 不想买千篇一律的冰箱贴
- ¥58-98 的「遇见」系列正中——便宜、有故事、轻便
- 回家后可能在小红书发一条："在根河买到了最好买的文创"

### 画像二：手账/文具圈
- 已经有 3 本以上手账、几十卷胶带
- 对"有叙事功能的文具"有执念
- 「猎人的标记」胶带和「树皮上的时间」手账会让他们兴奋
- 在小红书/手账社群有影响力——一个人买 = 十个人看到

### 画像三：生活方式消费者
- 25-35 岁，有独立收入，追求日常品质感
- 买香薰蜡烛、帆布袋、陶瓷杯、胸针——不是刚需，是"我想要这个"
- 「日常」系列的核心客群
- 愿意为故事多付 30% 的溢价

### 画像四：收藏者/送礼者
- 买给自己："我想要一件真正有手工艺价值的东西"
- 买给别人：重要的人的生日/毕业/升职礼物
- 「根河」系列的目标——¥328-498 是"贵但有分量"的礼物区间
- 限量编号 + 收藏证书 = 送礼的说服力

## 一句话销售话术

> **对游客说**：你在敖鲁古雅看到的驯鹿和白桦林，这三样东西帮你带回去。
>
> **对手账圈说**：这不是装饰胶带，是鄂温克猎人的森林导航系统。
>
> **对送礼的人说**：不用纠结尺寸颜色。这是在送一片森林。
>
> **对所有消费者说**：鹿隐卖的不是文创产品，是鄂温克人留在森林里的痕迹。
