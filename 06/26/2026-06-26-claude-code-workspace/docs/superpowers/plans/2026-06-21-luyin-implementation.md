# 鹿隐 LUYIN 文创礼盒 — 执行计划

> **Goal:** 将设计文档转化为可交付的品牌资产、产品文件、包装文件和文案，具备打样和生产条件。

**Architecture:** 6 个阶段递进——先锁定品牌基础（Logo/色彩/符号），再出产品设计文件（从成本最低的遇见系列开始打样验证），然后包装结构设计，最后整合文案和供应商 handoff。

**Deliverable Type:** 设计文件（AI/PSD/Figma）+ 规格书 + 文案文档

---

## 全局约束

- 品牌名：鹿隐 LUYIN，全程统一
- 色彩：桦皮白 #F5F0E8 / 深木棕 #5A3A20 / 苔原绿 #4A6A3A / 暖驼色 #C4844A / 浆果红 #8B3A2A
- Logo：线稿鹿角风格
- 四大符号：驯鹿角、白桦眼纹、撮罗子形、鄂温克纹样 —— 所有产品复用
- 三层产品线：遇见（¥58-98）、日常（¥168-238）、根河（¥328-498）
- 包装叙事：猎人的行囊 / 鄂温克人的桦皮箱 / 萨满的宝藏匣
- 共用元素：故事卡 + 干苔藓 + 皮绳捆扎，全程无塑料
- 设计哲学：每件产品的故事来自鄂温克的生活动作，不是贴文案

## Phase 7: 生产 Handoff

### Task 7.1: 供应商交付清单

**Files:**
- Create: `handoff/supplier-briefs.md`

**Deliverable:**
- 按供应商类型分组的交付物清单：
  1. **印刷供应商**（遇见全系 + 故事卡 + 包装纸）：AI/PDF 文件 + 纸张规格 + 数量
  2. **陶瓷供应商**（蜡烛杯 + 马克杯）：丝印文件 + 釉色样 + 模具要求
  3. **纺织品供应商**（帆布袋 + 布面手账）：丝印文件 + 面料样 + 缝制规格
  4. **金属制品供应商**（胸针 + 书签）：蚀刻/填色文件 + 金属材质 + 表面处理
  5. **木制品供应商**（扩香木 + 木扣）：3D 尺寸图 + 木材种类 + 表面处理
  6. **皮革供应商**（手账封面 + 皮绳）：皮料规格 + 染色要求 + 压印文件
  7. **桦树皮手工艺**（手工盒）：规格说明 + 品控宽容度说明
  8. **蜡烛/精油**：调香 brief + 蜡体规格 + 精油配方
- 每种供应商标注：最小起订量（MOQ）+ 预估交期 + 样品费用

**Review Gate:** 每种类型至少有 2 家备选供应商（除桦树皮手工艺可 1 家）。

---

### Task 7.2: 打样验收清单

**Files:**
- Create: `handoff/sample-checklist.md`

**Deliverable:**
- 每件产品的验收维度（打钩表格式）：
  - 色彩还原（对照色值，ΔE ≤ 3）
  - 材质触感（对照规格书中的手感描述）
  - 尺寸公差（标注每件产品的可接受公差）
  - 印刷/蚀刻清晰度（最小线宽 ≥ 0.3mm 处无断线）
  - 组装/结构完整性（开合、折叠、拼接等动作测试）
  - 故事元素完整性（故事卡、文案位置、谚语核对）
- 一次打样全检流程：
  1. 逐项检查打样
  2. 拍摄实物照片（自然光 + 棚拍）
  3. 标注不合格项，注明是否可接受或需重打
  4. 确认后可进入大货生产

**Review Gate:** 验收清单覆盖设计文档中每个产品的每个关键特性。

## 执行顺序建议

```
Phase 1 (品牌基础) ──→ Phase 2 (遇见系列) ──→ 打样验证 ──→ 调整
                         ↓
                    Phase 5 (包装设计，与 Phase 2 同步)
                         ↓
                    Phase 3 (日常系列) ──→ Phase 6 (文案)
                         ↓                      ↓
                    Phase 4 (根河系列) ──→ Phase 7 (Handoff)
```

- Phase 2 先行：成本低，验证整体方向
- Phase 5 与 Phase 2 同步：遇见系列的包装直接拿去打样
- Phase 3+4 在 Phase 2 打样反馈后启动，避免全线返工
- Phase 6（文案）可在 Phase 3 启动时并行
