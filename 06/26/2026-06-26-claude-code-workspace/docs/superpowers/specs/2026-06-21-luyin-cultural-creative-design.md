# 鹿隐 LUYIN — 敖鲁古雅文创礼盒设计文档

> 日期：2026-06-21
> 状态：设计定稿

---

## 一、项目概述

**品牌名**：鹿隐 LUYIN
**文化锚点**：敖鲁古雅（内蒙古根河）—— 中国最后的驯鹿部落，鄂温克人
**产品形态**：实体文创礼盒，三层价位
**销售渠道**：景区在地 + 线上电商（小红书/淘宝）
**设计哲学**：每件产品不是贴上去的文案，而是本身就来自一种鄂温克的生活动作。

---

## 二、品牌视觉体系

### 品牌名

**鹿隐**（LUYIN）— 驯鹿隐于白桦林，含蓄有诗意。两字好记，中英文友好。

### Logo

线稿鹿角风格。驯鹿角线条 + 文字，简洁但有辨识度。

### 色彩体系

| 色名 | 色值 | 用途 |
|------|------|------|
| 桦皮白 | #F5F0E8 | 主背景色 |
| 深木棕 | #5A3A20 | 文字/线条 |
| 苔原绿 | #4A6A3A | 点缀/插画 |
| 暖驼色 | #C4844A | 驯鹿/皮革元素 |
| 浆果红 | #8B3A2A | 点缀/火漆/印章 |

轻量线偏暖色系（森林叙事），中端线偏冷灰调（极简），共用同一套色板。

### 四大核心视觉符号

1. **驯鹿角** — 主视觉符号，可几何化、可写实、可线稿
2. **白桦眼纹** — 桦树皮上的黑色横纹，自然纹理
3. **撮罗子形** — 锥形帐篷，三角形 = 家、篝火、庇护
4. **鄂温克纹样** — 几何纹抽象化（菱形、云纹、鹿角纹）

## 三、产品线

### 🌿 轻量系列 ·「遇见」

**定价**：¥58-98
**渠道**：景区冲动消费 + 线上引流
**叙事**：给第一次听说敖鲁古雅的人——三件小物，三个画面，快速建立认知

| 产品 | 故事 |
|------|------|
| 「白桦林日记」A6 手绘线圈本 | 封面是驯鹿穿过晨雾的白桦林——这是鄂温克人每天醒来看到的画面。四种封面：晨牧、迁徙、撮罗子炊烟、星野下的鹿铃。 |
| 「猎人的标记」和纸胶带 ×2 | 鄂温克猎人深入大兴安岭时在白桦树上刻标记——指向猎物、标注水源。一卷「方向」（鹿角纹样），一卷「记号」（白桦眼纹）。撕一段 = 纸上的猎痕。 |
| 「鄂温克的六个瞬间」明信片 ×6 | 六个手绘生活切片：驯鹿饮水、老人削桦树皮、撮罗子升烟、迁徙过河、萨满鼓声、北极星下的鹿群。背面配鄂温克谚语。 |

| 产品 | 故事 |

| 「大兴安岭的呼吸」桦木香薰蜡烛 | 复刻大兴安岭深秋气味：白桦树汁清甜、松针凛冽、苔藓潮湿。木质烛芯燃烧发出篝火般的噼啪声。容器哑光陶瓷，刻极简鹿角线稿。 |
| 「撮罗子的早晨」几何鹿角杯 | 哑光釉面，杯身极简卧鹿线条。鄂温克谚语："驯鹿卧下，就是家"。两色：桦皮白、苔原绿。手握时盖住鹿身，只露出角。 |
| 「迁徙」帆布袋 | 16安帆布，丝印鄂温克季节性迁徙路线的抽象地图。内置口袋印："家是移动的，森林是永恒的"。 |

| 「你是谁？」珐琅胸针 ×3 | 鹿角（领路的）、撮罗子（筑巢的）、白桦叶（生长的）。鄂温克人相信每个灵魂都有对应的动物或树木——这枚胸针是你的森林身份。 |

**定价**：¥328-498
**渠道**：线上限量 + 收藏级送礼
**叙事**：给已经爱上敖鲁古雅的人——不是消费品，是可收藏的物件

| 产品 | 故事 |
|------|------|
| 「森林的容器」桦树皮手工盒 | 鄂温克匠人手工制作——真桦树皮剥离、蒸煮、压平、缝合。每块树皮纹理独一无二。鄂温克人用桦树皮做船、做容器、做摇篮——这个盒子是森林的容器。 |
| 「皮绳上的四季」植鞣皮手账 | 意大利植鞣皮，手工染暖驼色。压印鹿角。会随使用变色包浆，越用越像旧物。可替换内芯，终身使用。鄂温克人相信万物有灵——一本跟你一起老去的手账。 |
| 「鹿角的方向」黄铜书签 | 实心黄铜蚀刻镂空，鹿角分叉造型。随时间氧化成深棕金。猎人通过鹿角朝向判断鹿群方向——这枚书签指向你上次停下的那一页。 |

| 「最后的森林」限量艺术微喷 | 合作插画师敖鲁古雅驻地创作，限量编号，售完不加印。附收藏证书，设计为"森林通行证"风格。 |

## 四、包装与开箱体验

**核心理念**：盒子不是容器，是故事的第一页。打开的动作本身，就是在走进敖鲁古雅。

### 遇见：「猎人的行囊」

- 牛皮纸盒，线稿森林轮廓印刷
- 桦树皮纹理腰封 —— 撕开 = 猎人在树上刻第一道标记
- 木丝填充，像森林地表
- 底部折叠成敖鲁古雅手绘地图
- 地图上印：「每个鄂温克猎人心里都有一张森林的地图。这张是你的。」

### 日常：「鄂温克人的桦皮箱」

- 桦树皮纹理硬盒 + 皮绳捆扎
- 解开皮绳 = 在营地安顿下来
- 分区托盘，每个格子印鄂温克词：oron（驯鹿）、juu（家）、kure（森林）
- 盒子可重复使用作收纳
- 盒底印：「鄂温克人搬家时，把整个生活打包进一个桦树皮箱子。这个，到你手里了。」

### 根河：「萨满的宝藏匣」

- 真正桦树皮贴面硬盒，皮绳 + 鹿角形木扣
- 浆果红火漆封印（鹿角图案）
- 四层逐层深入：
  1. 开盖 → 限量微喷 + 收藏证书
  2. 上层托盘 → 皮手账 + 铜书签 + 压干白桦叶
  3. 底层 → 桦树皮手工盒 + 扩香木 + 干苔藓
- 每层间夹一张纸条：
  - 「你听见鹿铃了。」
  - 「脚下是苔藓，很软。」
  - 「抬起头，你已经在森林深处。」
  - 「这个箱子本该装驯鹿铃铛和茶叶。现在你装什么，它就是什么。」

### 三层共用元素

- **故事卡**：正面 = 产品对应的鄂温克文化片段，背面 = 使用场景提示
- **干苔藓一小束**：大兴安岭真实苔藓，干燥处理。"这是驯鹿的食物，也是你脚下森林的碎片。"
- **皮绳/麻绳捆扎**：全程无塑料封膜
